StriveForge is a modern and minimalist productivity web application built to help users develop discipline, consistency, and strong daily habits. In today’s fast-paced digital world, staying focused and maintaining routines is difficult, so StriveForge is designed as a simple yet powerful solution that encourages self-improvement through structured habit tracking. The application allows users to track key daily activities such as gym workouts, coding practice, and reading sessions, helping them stay accountable to their personal goals. Each completed activity contributes to a streak system, which motivates users to maintain consistency without breaking their progress chain. Along with habit tracking, StriveForge includes a focus timer based on the Pomodoro technique, which helps users improve concentration by dividing work into focused intervals with short breaks. This makes studying, coding, or working out more effective and less overwhelming. The application also features a productivity score system that gives users a visual representation of their daily performance, making self-evaluation easier and more engaging. The interface is designed with a clean, dark, and modern UI approach to reduce distractions and provide a smooth user experience. Built using HTML, CSS, and JavaScript, along with LocalStorage for saving user data in the browser, StriveForge works without any backend while still providing a functional and interactive experience. The main goal of this project is to combine simplicity with productivity tools in a way that helps users build better habits, stay focused, and gradually improve their lifestyle through consistent daily effort.
